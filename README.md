# kind-vege

A zero-waste, plant-based lifestyle platform for Minjoo Cho (Kind Vege).

## Features
- **Video of the Week**: Featured sustainability insights.
- **Vegan Locator**: Global plant-based restaurant finder using Geolocation API.
- **Social Hub**: Curated links to Instagram, YouTube, TikTok, and Naver.
- **Minjoo Cho Identity**: Highlights her role as an author, chef, and environmental educator.

## Development
```bash
npm install
npm run dev
```
